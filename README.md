This is a laravel based e commerce project.
The project has admin and user roles.
Features in the project are :- a) User can view products b) add to cart c) order products d) a real time chat system e) Admin can add,editand delete product .
Tech used - a) MySql (Database) b) frontend (html , css , javascript - api call with ajax , js full calaender) c) backend (laravel) , firebase(for real time chat)
